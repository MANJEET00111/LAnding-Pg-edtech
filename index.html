HTML
Html is standard markup language used to create the structure and content of Web pages. It consists of a set of elements known as tags which define the different part of a web page such as heading paragraph, images, links, forms and more . Html provides the basic framework for displaying content on the web and the server as the backbone of every web page.
CSS
Css is a style sheet language used to control the presentation and the layout of the stm document. It allows web designer to design how element should be displayed on a web page including its properties colours fonts spacing etc. By separating the content from its presentation developer can create visual appealing and consistent website across different devices and screen sizes .
